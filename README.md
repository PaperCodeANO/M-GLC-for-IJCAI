# M-GLC-for-IJCAI

## IJCAI Review Repository

This repository is created **exclusively for the purpose of IJCAI peer review**.

It contains the code and supplementary materials necessary to reproduce the experimental results reported in the submitted manuscript.  
The repository is intended **only for anonymous review and verification**, and is **not an official public release**.

After the review process is completed, this repository may be:
- updated with cleaned code and documentation, or  
- merged into an official public repository, or  
- taken offline, depending on the final publication decision.

If you have any questions or encounter issues when running the code, please refer to the instructions provided in this repository.

Thank you for your time and consideration.
